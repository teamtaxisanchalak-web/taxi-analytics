# 🚕 Taxi Sanchalak — Analytics Hub

A tabbed dashboard launcher hosting all AI analytics reports on **GitHub Pages**.

---

## 📁 Files in this repo

```
index.html                  ← Main tabbed launcher (open this)
dashboard.html              ← Command Center (Booking + Recharge)
live_dashboard.html         ← Call Analytics Live Dashboard
monitoring_dashboard.html   ← Ops Monitor v4.2
```

---

## 🚀 How to deploy on GitHub Pages (Step by Step)

### Step 1 — Create a GitHub account
1. Go to **https://github.com**
2. Click **Sign up** and create your account

### Step 2 — Create a new repository
1. Click the **+** icon (top right) → **New repository**
2. Name it: `taxi-analytics` (or any name you like)
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload all files
1. In your new repo, click **Add file → Upload files**
2. Upload ALL 4 files:
   - `index.html`
   - `dashboard.html`
   - `live_dashboard.html`
   - `monitoring_dashboard.html`
3. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to your repo **Settings** (top tab)
2. Scroll to **Pages** (left sidebar)
3. Under **Source** → select **Deploy from a branch**
4. Branch: **main** | Folder: **/ (root)**
5. Click **Save**

### Step 5 — Get your live URL
After ~1 minute, your site will be live at:
```
https://YOUR-USERNAME.github.io/taxi-analytics/
```
